--- title: Gestion de projet (agile) nav_order: 3 --- # Gestion de projet (agile) ## Approche - Itérative / incrémentale par **modules** (Renew → TBH7 → Public). - Branches Git : - `dev` : développement libre. - `int` : intégration E2E (sandbox HA, base INT). - `main` : production (stabilisée). - Revue via PR / bundles git si besoin. ## Environnements - **DEV** (local) : `.env`, Mongo locale (ou Docker), HelloAsso **stubbé** si souhaité. - **INT** (VPS) : `.env` dédié, `HELLOASSO_ENV=sandbox`, logs, données de test. - **PROD** (VPS) : `.env` dédié, `HELLOASSO_ENV=prod`. ## Definition of Done (phase Renew) - Lien de renouvellement fonctionne (sièges, tarifs visibles). - Choix tarifs + champs justificatifs conditionnels. - Checkout HelloAsso (sandbox INT / prod PROD). - Logs & exports OK.
